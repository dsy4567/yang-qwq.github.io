              # 保护石插件简易教程

              由于我们资金不足，买不起常见的圈地插件

              所以，我们选择了开源且免费的ProtectionStones (opens new window)插件

              # 获取保护石

              要获取保护石，请在打开菜单后依次点击

              或者直接执行指令以获取保护石

              提示

              不同的保护石类型代表了不同的价格，激活消耗以及保护大小，详细请查看保护石价格概览

              /ps get [block]

              如果你没有输入block的类型，你将会收到一个简单的菜单，你可以通过点击文字直接获取保护石，例如

              ===== 保护石 (点击获取) =====
> 16x (16 x 16) ($160)
> 32x (32 x 32) ($160)
> 64x (64 x 64) ($160)

              # 放置保护石

              这一步很简单，如果一切正常，你会收到扣费通知以及一个

              此区域现已被保护

              的一条消息通知，这说明你的领地已经注册成功了！

              # 查看领地信息

              执行该命令后，你会收到一长串信息，具体解释如下

              /ps info

              例如：

              ===== 领地信息 =====
Region: ps-114x51y4z, Priority: 0
Type: 32x
Flags: invincible: -g MEMBERS ALLOW, greeting: 正在进入 Yang_161941 的保护区域, pvp: DENY, tnt: DENY, chest-access: -g NON_MEMBERS DENY, fire-spread: DENY, greeting-action: 正在进入 Yang_161941 的保护区域, notify-leave: -g OWNERS true, use: ALLOW, farewell-action: 正在离开 Yang_161941 的保护区域, creeper-explosion: DENY, notify-enter: -g OWNERS true, farewell: 正在离开 Yang_161941 的保护区域, ps-home: -1383.0 73.0 -771.0, .
Owners: Yang_161941
Members: (没有协助成员)
Bounds: (-1415, -803) -> (-1351, -739)

              其中，Region表示的是你的领地id（如果你已经改名将会显示为<名称> (<领地id>)）

              Type栏内显示的是保护石的类型

              Flags栏内显示的是当前领地的配置（本插件依赖WorldGuard，配置项可以非常轻松的前往WorldGuard文档 (opens new window)查看）

              Owners，Members指示的是当前领地的主人以及协作者

              最后的Bounds指的是领地保护的范围

              提示

              一个领地可以有多个主人，主人可以自己修改领地配置

              # 设置领地名称（可选）

              现在该给你的领地取个新名字了吧！

              注意

              领地的名字在服务器内是独一无二的，设置时请避免重复名字

              要设置名字，请执行

              /ps name <新名字>
              想要取消设置名字？

              请执行

              /ps name none

              # 分享领地

              想和服务器的其他人分享你的领地？不用慌！

              首先，你需要获取你的领地ID/名字（可以通过计分板查看）

              然后告诉他（她）们领地ID/名字

              收到领地ID/名字后执行下面的命令后，你就可以传送至目标领地了！

              /ps tp <ID/名字>

              # 配置领地

              /ps flag [flagname] [var|default|null]

              配置项请前往WorldGuard文档 (opens new window)查看定义
